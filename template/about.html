    The project aims and objectives will be achieved after completion of this project are:
    1. Providing a common coding platform to all the coders and developers.
    2. Providing an online compiler/Interpreter that runs on almost every device (i.e., Laptop, Desktop, Mobile, Tablet etc.).
    3. Making cloud-based portable compiler available to everyone. No longer need to download SDK for different language, no more issues of setting path variables, no need of storage space on user’s system, no more mess.
    4. Providing special stage for teachers, to post questions, and students, to solve that within time, so that students can get the coding ideas quickly.
    5. Making online learning more interactive and easier.
    6. Making competitive coding much easier and available to everyone.
    7. Special Winner Board for students, who solved the question at less time and more efficiently.

BACKGROUND OF PROJECT:

      Our project, Virtual Programming Quiz Platform, provides the environment to execute particular program of particular language (i.e. C, C++, PYTHON, JAVA). This web based project has an built-in online compiler that allows user to compile user’s code and run it in an online terminal. The project can also use to check that the program working properly or not using test case, submitted by the user. There are three types of users in this program,
      Admin
      • The Administrator have the supreme authority in this program. An admin can create or delete any other user’s account or data as he/she has the total access to the program database.
      Teacher
      • The Teacher or staff user, can post coding related questions and review the answers, submitted by the Students, and by reviewing the student’s completion time and efficiency, he/she can pick the best answer given by the student and make him/her appear in the Winner Board.
      Student
      • The Student or active user, is the base of this program. They can pick any of the posted question, go through it and solve it through coding. There would be given an option to select the coding language in which they want to write and compile their code. The application will compile it and return the output in a terminal.
