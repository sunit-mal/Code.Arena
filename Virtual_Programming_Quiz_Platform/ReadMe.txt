● My project, Code Arena, provides the environment to execute program in particular language (i.e., C, C++, 
PYTHON, JAVA). This web-based project has a built-in online compiler that allows user to compile user’s code and run it in an 
online terminal. The project can also use to check that the program working properly or not using test case, submitted by the user.

● Tech Stack: Python, Django Backend, MVT Architecture, SQLite Database, HTML, CSS, JavaScript frontend
